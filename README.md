# 🍽️ Zomato Data Analysis Dashboard

An interactive Excel dashboard built using Zomato restaurant data to analyze customer preferences, restaurant performance, cuisines, ratings, and geographical trends. The dashboard provides actionable insights through dynamic visualizations, filters, and summary metrics.

---

## 📌 Project Objectives

- Analyze restaurant distribution across different cities.
- Identify the most popular cuisines and restaurant categories.
- Study customer ratings and voting patterns.
- Compare restaurant performance using key business metrics.
- Create an interactive dashboard for quick data exploration and decision-making.

---

## 📂 Dataset

The dataset contains restaurant information from Zomato, including:

| Column | Description |
|--------|-------------|
| Restaurant Name | Name of the restaurant |
| City | Location of the restaurant |
| Cuisine | Type of cuisine served |
| Rating | Average customer rating |
| Votes | Number of customer votes |
| Price Range | Cost category |
| Online Delivery | Availability of online delivery |
| Table Booking | Availability of table booking |
